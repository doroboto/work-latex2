## 개요

**HTP50cork**는 우주항공 분야에서 사용되는 **코르크 기반 열차폐재(TPS, Thermal Protection System)** 의 열 응답을 분석하기 위한 시뮬레이션 도구입니다. 코르크(P50) 층과 금속 기판(substrate) 의 1차원 비정상 열전달을 유한 차분법(Finite Difference Method)으로 풀며, **열분해에 의한 물성 변화**를 고려합니다.

### 주요 특징

- **Mathematica 기반 검증 모델의 Python 구현** — Mathematica의 열전달 흐름(Math-Flow)을 Python(SciPy)으로 재구현
- **열분해(Pyrolysis) 이중 모드** — Simple 모드 / Advanced 모드 지원 (Density Change 토글)
- **Apple-style Dark UI** — 영문 인터페이스, Apple 디자인 시스템 색상 팔레트 (Segoe UI 폰트)
- **실시간 진행률 표시** — 별도 스레드에서 시뮬레이션 실행, 진행률 바로 경과 확인
- **3개 그래프 동시 시각화** — 온도 이력, 비열(Cp) 변화, 열분해 비교 그래프
- **CSV/PNG 내보내기** — 시뮬레이션 결과를 데이터 및 이미지로 저장
- **사용자 기본값 영구 저장** — 커스텀 재료 데이터를 로컬에 영구 저장
- **커스텀 알림음** — `winsound.Beep` 기반 음악적 알림 (Windows 시스템 경고음 대체)

## 물리 엔진 상세

### 지배 방정식

1차원 비정상 열전달 방정식(에너지 보존):

```
ρ(T) · Cp(T) · ∂T/∂t = ∂/∂x [k(T) · ∂T/∂x]
```

- **ρ(T)**: 밀도 [kg/m³]
- **Cp(T)**: 비열 [J/(kg·K)]
- **k(T)**: 열전도도 [W/(m·K)]

### 공간 이산화 (Finite Difference)

구조는 두 영역으로 구성됩니다:

```
  Cork (P50)           Metal
|←—— L1 ——→|←——— L2 ———→|
|   N1 nodes  | m |  N2 nodes  |
  dx1 간격     인터페이스   dx2 간격
```

- **노드 0**: 표면(Surface) — 대류/복사 경계 조건
- **노드 1 ~ m-1**: 코르크 내부
- **노드 m**: 코르크-금속 인터페이스 (가중 평균 열용량)
- **노드 m+1 ~ N-2**: 금속 내부
- **노드 N-1**: 바닥(Bottom) — 단열(Adiabatic) 경계 조건

### 경계 조건

| 위치 | 조건 | 설명 |
|------|------|------|
| **표면 (x=0)** | 대류 | `q_conv = h(t) · [Tr(t) - T_surface]` |
| **바닥 (x=L1+L2)** | 단열 | `∂T/∂x = 0` (열유속 = 0) |
| **인터페이스 (x=L1)** | 연속 | 코르크/금속 물성의 가중 열용량으로 계산 |

### 노드 간 열전도

인접 노드 간 유효 열전도도는 **조화 평균**:

```
k_eff = 2·k₁·k₂ / (k₁ + k₂)
```

---

### 열분해(Pyrolysis) 모델링

열분해 모델은 두 가지 모드를 제공하며, 역할이 명확히 분리되어 있습니다:

| 역할 | 담당 |
|------|------|
| **k, Cp의 온도 의존성** | 코르크 물성 테이블 (보간) |
| **k, Cp의 비가역 고정** | Cp/k Freeze 로직 (Simple & Advanced 공통) |
| **ρ의 열분해 감소** | 질량 프로파일 → `ρ = ρ_virgin × mass_func(Tmax)/100` (Advanced만) |

#### 상태 변수: Tmax (최대 도달 온도 이력)

각 노드마다 **지금까지 도달한 최대 온도(Tmax)** 를 ODE 상태 변수로 추적합니다:

```
dTmax/dt = max(0, dT/dt) · σ(T - Tmax)
```

σ(x)는 시그모이드 함수(stiffness=100)로, Tmax는 온도가 상승할 때만 증가하고 냉각 시 고정됩니다.

---

#### Simple 모드 — Cp/k 비가역 고정

**사용 파라미터**: `T_critical` (기본: 500°C)

```
factor = σ(Tmax - T_critical) · σ(Tmax - T_current)
Cp = Cp(T) · (1 - factor) + Cp(Tmax) · factor
k  = k(T)  · (1 - factor) + k(Tmax)  · factor
```

**두 개의 시그모이드 게이트의 의미:**

| 게이트 | 조건 | 역할 |
|--------|------|------|
| `σ(Tmax - T_critical)` | Tmax > T_critical | "열분해가 시작되었는가?" |
| `σ(Tmax - T_current)` | Tmax > T_current | "현재 냉각 중인가?" |

**동작 원리:**
- **가열 중** (T ≈ Tmax): 두 번째 게이트 ≈ 0.5 → factor가 작아 테이블 물성 그대로 사용
- **T_critical 미만**: 첫 번째 게이트 ≈ 0 → 보정 없음
- **T_critical 초과 + 냉각 중**: factor ≈ 1 → **Cp, k를 Tmax 시점 값으로 고정** (비가역)

시그모이드를 사용하는 이유: `if-else` 불연속성을 피하고 ODE 솔버의 수치 안정성을 유지하기 위함.

---

#### Advanced 모드 — Simple + 밀도 감소

**사용 파라미터**: `T_critical` (Simple과 동일) + 질량 프로파일 (Fig.1)

Advanced 모드는 Simple 모드의 **모든 로직을 포함**하고, 추가로 **질량 프로파일 기반 밀도 보정**을 적용합니다:

**Step 1: Cp/k 비가역 고정 (Simple과 동일)**

```
factor = σ(Tmax - T_critical) · σ(Tmax - T_current)
Cp = Cp(T) · (1 - factor) + Cp(Tmax) · factor
k  = k(T)  · (1 - factor) + k(Tmax)  · factor
```

**Step 2: 밀도 보정 (Advanced 추가)**

```
mass_pct = mass_func(Tmax)                    -- Fig.1 보간 (100% → ~20%)
ρ = ρ_virgin × mass_pct / 100                 -- 고정 격자 → 질량 감소 = 밀도 감소
```

고정 격자(fixed grid) 모델에서 **부피는 변하지 않으므로**, 질량 변화가 직접 밀도 변화로 이어집니다:

```
ρ = 질량 / 부피  (부피 고정)  →  Δρ/ρ = Δm/m
```

**Tmax 기반**이므로 비가역성이 자동 보장됩니다 — 냉각 시에도 Tmax가 고정되어 밀도가 다시 올라가지 않습니다.

**밀도 변화 예시:**

| Tmax | mass_pct (Fig.1) | ρ = 465.6 × mass/100 | 의미 |
|------|:-:|:-:|------|
| 26°C | 100% | **465.6** kg/m³ | 원시 코르크 |
| 226°C | 85% | **395.8** kg/m³ | 초기 분해 |
| 426°C | 50% | **232.8** kg/m³ | 상당 분해 |
| 506°C | 24.5% | **114.1** kg/m³ | 거의 완전 탄화 |
| 1376°C | 20% | **93.1** kg/m³ | 완전 탄화 |

---

#### Simple vs Advanced 비교 요약

| 항목 | Simple | Advanced |
|------|:------:|:--------:|
| **Cp 비가역 고정** | ✅ | ✅ (동일) |
| **k 비가역 고정** | ✅ | ✅ (동일) |
| **ρ 질량 프로파일 보정** | ❌ (테이블 ρ 그대로) | ✅ `ρ = ρ_virgin × mass/100` |
| **필요 파라미터** | T_critical | T_critical + 질량 프로파일 |
| **물리적 설명 수준** | Cp/k 변화만 | Cp/k 변화 + 질량 손실에 의한 밀도 감소 |

> **설계 철학**: 물성 테이블에는 측정된 열물성(k, Cp)만 담고, 열분해의 비가역적 효과(Cp/k 고정, 밀도 감소)는 코드 로직으로 분리합니다. 이로써 데이터와 로직의 **이중 적용**을 방지합니다.

| 항목 | 값/방법 |
|------|---------|
| **ODE 솔버** | SciPy `solve_ivp` — `BDF` (Backward Differentiation Formula) |

| `N1` | 5 | 코르크 유한 차분 노드 수 |
| `N2` | 5 | 금속 유한 차분 노드 수 |
| `t_final` | 120 s | 시뮬레이션 종료 시간 |
| `T_init` | 63 °C | 초기 온도 |

#### 3. Pyrolysis (Cork P50) — 열분해 설정

| 파라미터 | 기본값 | 사용 모드 | 설명 |
|----------|--------|-----------|------|
| `T_critical` | 500°C | Simple & Advanced | 열분해 비가역 고정 활성화 임계 온도 |

**Density Change 토글 ON** (기본값 — Advanced 모드):
- Simple과 동일한 Cp/k 고정 + 질량 프로파일(Mass Profile) 기반 밀도 보정
- Mass Profile 행이 토글 아래에 표시됨

**Density Change 토글 OFF** (Simple 모드):
- Cp/k를 Tmax 시점 값으로 고정 (냉각 시 비가역)
- Mass Profile 행 숨김

### 시뮬레이션 실행

1. 파라미터와 데이터를 설정한 뒤 **RUN SIMULATION** 버튼 클릭
2. 진행률 바가 시뮬레이션 경과를 실시간 표시
3. **두 개의 솔버가 동시 실행**:
   - **Pyrolysis 포함**: 열분해 효과가 반영된 시뮬레이션
   - **Pyrolysis 미포함**: 비교를 위한 기준(baseline) 시뮬레이션
4. 완료 시 상태 표시줄에 소요 시간이 표시되고 3개 그래프가 자동 업데이트

### 그래프 영역

| 그래프 | 위치 | 내용 |
|--------|------|------|
| **Temperature History** | 상단 | 선택한 노드의 시간-온도 이력 |
| **Specific Heat (Cp)** | 중단 | 선택한 노드의 시간-Cp 변화 (열분해 영향 확인) |
| **Pyrolysis Comparison** | 하단 | 열분해 포함(실선) vs 미포함(점선) 온도 비교 |

각 그래프에는 다음 버튼이 있습니다:
- **Peak Temp** (온도 이력 그래프만): 각 모니터링 노드의 최대 온도와 도달 시간 팝업
- **CSV**: 해당 그래프의 데이터를 CSV 파일로 내보내기
- **PNG**: 해당 그래프를 밝은 테마 고해상도(300 DPI) PNG로 내보내기
- **Options**: 표시할 노드, 색상, 레이블을 설정하는 팝업 창

#### 그래프 옵션의 노드 지정 방법

| 입력 | 설명 |
|------|------|
| `0` | 표면(Surface) 노드 |
| `surf`, `cork` | 표면 노드 (= 0) |
| `mid`, `int`, `metal` | 인터페이스 노드 (= N1) |
| `bot` 또는 `-1` | 가장 마지막(바닥) 노드 |

### 데이터 내보내기

- **CSV 내보내기**: 시간 열 + 각 모니터링 노드의 온도(또는 Cp) 데이터
- **PNG 내보내기**: 밝은 배경의 고해상도 이미지 (논문/보고서용)

## 기본 데이터셋

### 코르크(P50) 물성 데이터 출처

> Sakraker et al., *CEAS Space Journal* (2022), DOI: [10.1007/s12567-021-00395-z](https://doi.org/10.1007/s12567-021-00395-z)

- **열전도도(k)**: Fig.3 평균 곡선 기반
- **비열(Cp)**: Fig.2 평균 곡선 기반 (논문 단위 J/(g·K) → J/(kg·K) 변환)
- **정규화 질량**: Fig.1 곡선 (열분해 진행도 프로파일)
- **밀도(ρ)**: 초기 밀도 465.6 kg/m³로 일정 (밀도 변화는 Advanced 모드에서 질량 프로파일로 계산)
- **온도**: K → °C 변환
- **Cork (No Pyro)**: k는 최소값(0.062) 일정, Cp는 누적 최대값(running maximum)으로 처리

### 금속 기판 기본 데이터

> **Al 7075-T6** 열물성 (ρ=2810 kg/m³, k≈130–170 W/(m·K), Cp≈960–1200 J/(kg·K))

### 기본 데이터 범위

| 데이터 | 온도/시간 범위 | 비고 |
|--------|-----------|------|
| Cork | 26.85 ~ 2726.85 °C | 26개 포인트, ρ=465.6 고정 |
| Cork (No Pyro) | 26.85 ~ 2726.85 °C | k=0.062 일정, Cp 누적 최대값 (비교용) |
| Mass | 26.85 ~ 2726.85 °C | 100% → 20% 정규화 질량 |
| Metal (Al 7075-T6) | 26.85 ~ 626.85 °C | 7개 포인트, ρ=2810 kg/m³ |
| h(t) | 0 ~ 120 s | 최대 500 W/m²K |
| Tr(t) | 0 ~ 120 s | 최대 ~977°C |

## 사용자 기본값 저장

사용자가 수정한 재료/경계 데이터는 **로컬 JSON 파일**에 영구 저장할 수 있습니다:

```
%LOCALAPPDATA%\HTP50cork\user_defaults.json    (Windows)
~/HTP50cork/user_defaults.json                  (macOS/Linux)
```

### 연동 저장 그룹 (Linked Core Keys)

`cork`, `cork_no_pyro`, `mass` 세 데이터는 **연동 그룹**으로 관리됩니다. 하나를 저장하면 나머지 두 개도 함께 저장되어 데이터 일관성을 보장합니다.

`metal`, `h`, `Tr`은 독립적으로 개별 저장됩니다.
