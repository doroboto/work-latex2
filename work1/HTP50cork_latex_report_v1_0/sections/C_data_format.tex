\chapter{(부록 C) 입력 데이터 형식(CSV)}

\section{데이터셋 개요}
프로그램은 재료 물성 및 경계조건을 CSV로 입력받는다.
기본적으로 각 CSV는 1행 헤더 + 수치 행으로 구성된다.

\section{단위 주의}
\begin{itemize}
  \item 온도: 기본 \si{\celsius} 기준(테이블과 GUI가 \si{\celsius}로 가정)
  \item $h$: \si{W/(m^2.K)}, $k$: \si{W/(m.K)}, $C_p$: \si{J/(kg.K)}, $\rho$: \si{kg/m^3}
  \item 시간: \si{s}
\end{itemize}
